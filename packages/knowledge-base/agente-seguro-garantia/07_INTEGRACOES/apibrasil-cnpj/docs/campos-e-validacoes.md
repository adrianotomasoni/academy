# Campos e Regras de Pré-qualificação (esqueleto)

> Preencher na implementação. Regras iniciais:

| Validação | Regra | Ação do agente |
|---|---|---|
| Situação cadastral | ≠ ATIVA | Bloqueio: informar irregularidade cadastral |
| Tempo de atividade | < 2 anos | Alerta: subscrição restritiva, possível exigência de contragarantias |
| CNAE vs objeto | Incompatível | Alerta: questionar aderência técnica |
| Capital/porte vs IS | IS >> porte | Alerta: provável redução/contragarantia; IS > R$ 20M → escalar |
