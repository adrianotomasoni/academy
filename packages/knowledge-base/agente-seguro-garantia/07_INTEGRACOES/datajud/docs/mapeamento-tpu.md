# Mapeamento TPU → Sub-modalidade (esqueleto)

> Preencher com códigos das Tabelas Processuais Unificadas (CNJ) conforme implementação.

| Segmento justiça (dígito J do nº CNJ) | Justiça | Grupo de sub-modalidade |
|---|---|---|
| 4 | Federal | Judicial Fiscal / Civil |
| 5 | Trabalho | Depósito Recursal / Judicial Trabalhista |
| 8 | Estadual | Judicial Civil / Fiscal (estadual) |

| Indicador de fase (classe/movimento) | Sub-modalidade indicada |
|---|---|
| Recurso ordinário/revista pendente (JT) | Depósito Recursal Trabalhista |
| Cumprimento de sentença / execução (JT) | Judicial Trabalhista |
| Execução fiscal (classe 1116) | Judicial Fiscal |
| Execução de título extrajudicial / cumprimento (cível) | Judicial Civil |
