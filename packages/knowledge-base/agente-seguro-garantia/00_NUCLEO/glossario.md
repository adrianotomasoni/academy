# Glossário de Seguro Garantia

| Termo | Definição |
|---|---|
| **Tomador** | Quem assume a obrigação garantida e contrata a apólice (devedor/contratado/réu). |
| **Segurado** | Beneficiário da garantia; quem é indenizado em caso de inadimplemento (credor/contratante/juízo). |
| **Seguradora** | Empresa regulada pela SUSEP que emite a apólice e assume o risco. |
| **IS (Importância Segurada)** | Valor máximo garantido pela apólice. |
| **Objeto** | Obrigação específica garantida (contrato nº X, processo nº Y, edital nº Z). |
| **Vigência** | Período de validade da apólice. Prorrogável por endosso. |
| **Endosso** | Alteração formal da apólice (prazo, valor, objeto) emitida pela seguradora. |
| **Prêmio** | Valor pago pelo tomador à seguradora pela emissão da apólice. |
| **Cláusula de incontestabilidade** | Impede que a seguradora questione a validade da garantia perante o segurado após a emissão. Obrigatória em apólices judiciais (Circular SUSEP 662/2022). |
| **Sinistro** | Caracterização do inadimplemento da obrigação garantida, comunicado formalmente pelo segurado. |
| **Expectativa de sinistro** | Comunicação prévia do segurado indicando possível inadimplemento (fase anterior à reclamação). |
| **Reclamação de sinistro** | Pedido formal de indenização pelo segurado. |
| **CCG (Contrato de Contragarantia)** | Instrumento pelo qual o tomador (e fiadores) se obriga a ressarcir a seguradora em caso de pagamento de sinistro. |
| **Contragarantia** | Garantias oferecidas pelo tomador à seguradora (aval dos sócios, fiança, bens). |
| **Limite de crédito** | Capacidade aprovada pela seguradora para o tomador emitir apólices. Avaliado por análise cadastral e financeira. |
| **Subscrição** | Processo de análise de risco da seguradora para aprovar a emissão. |
| **BID Bond** | Garantia de proposta em licitação. |
| **Performance Bond** | Garantia de execução contratual. |
| **Completion Bond** | Garantia de conclusão de obra/projeto, geralmente exigida por financiador. |
| **Depósito recursal** | Depósito exigido para interpor recurso na Justiça do Trabalho. Substituível por seguro garantia (Art. 899, §11 da CLT). |
| **Caução** | Garantia oferecida em processo judicial ou contrato. O seguro garantia é modalidade de caução (Art. 835, §2º do CPC). |
| **Trânsito em julgado** | Decisão judicial definitiva, sem possibilidade de recurso. Marco para baixa de apólices judiciais. |
| **Baixa de apólice** | Encerramento formal da garantia, mediante comprovação de extinção da obrigação. |
| **Lista CNJ** | Relação de seguradoras habilitadas a emitir seguro garantia judicial, mantida pelo CNJ. |
| **SUSEP** | Superintendência de Seguros Privados — órgão regulador do mercado de seguros. |
| **CNSP** | Conselho Nacional de Seguros Privados — órgão normativo superior. |
| **PGFN** | Procuradoria-Geral da Fazenda Nacional — regulamenta garantias em execuções fiscais federais. |
| **ACL** | Ambiente de Contratação Livre (mercado de energia). |
| **RJ (Recuperação Judicial)** | Regime de reestruturação de empresas em crise. Bloqueio padrão de subscrição. |
